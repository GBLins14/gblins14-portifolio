# GBLins Dev - Portfólio

## 📋 Descrição do Projeto

**GBLins Dev** é um portfólio profissional de desenvolvimento de sites, criado para apresentar os serviços especializados em desenvolvimento web para pequenos negócios e comércios locais. O site demonstra qualidade, profissionalismo e inovação através de uma interface moderna, responsiva e de alto desempenho.

## ✨ Características Principais

- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e dispositivos móveis
- **Performance Otimizada**: Sites rápidos e leves que garantem melhor experiência ao usuário
- **Layout Moderno**: Interface limpa e intuitiva com navegação fluida
- **Integração WhatsApp**: Botões CTA conectados direto ao WhatsApp para consultas
- **Portfolio Dinâmico**: Showcase de projetos anteriores como Wheeling House, Daysoltec e Adamis Construções
- **Tipografia Premium**: Uso da fonte Manrope do Google Fonts

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Estilos modernos e responsivos
- **JavaScript**: Interatividade e funcionalidades extras
- **Google Fonts**: Tipografia profissional
- **Responsive Design**: Mobile-first approach

## 🚀 Como Usar

### Abrir Localmente
1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em seu navegador web
3. Navegue pelas diferentes seções do site

### Estrutura das Seções

- **Header**: Navegação com links para as principais seções e botão de orçamento
- **Hero Section**: Apresentação principal com CTA para orçamento e projetos
- **Sobre (Seção 2)**: Três pilares dos serviços: Qualidade, Identidade Visual e Performance
- **Projetos**: Portfolio com 4 projetos realizados
- **Contato**: Link direto via WhatsApp

## 🎨 Customização

### Editar Estilos
Modifique o arquivo `assets/css/style.css` para alterar cores, fontes e layouts.

### Adicionar Novos Projetos
1. Adicione a imagem do projeto em `assets/images/`
2. Insira um novo card na seção de projetos do `index.html`

### Atualizar Informações de Contato
Procure pela URL do WhatsApp no `index.html` e atualize o número conforme necessário.
