# Adamis Construções Website

## Project Description
This project is a website for Adamis Construções, a construction company specializing in high-quality civil construction services. The website is designed to provide information about the company's services, recent projects, and contact details, all while adhering to a modern and responsive design inspired by the reference website.

## Project Structure
The project is organized as follows:

```
adamis-construcoes-website
├── assets
│   ├── css
│   │   └── style.css        # Styles for the website
│   ├── js
│   │   └── script.js        # JavaScript functionality
│   └── fonts                # Custom fonts for typography
├── images                   # Directory for images used in the website
├── index.html               # Main HTML file for the website
└── README.md                # Documentation for the project
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd adamis-construcoes-website
   ```

2. **Open the Project**
   Open the `index.html` file in your web browser to view the website.

3. **Modify Styles and Scripts**
   - Update the `assets/css/style.css` file to change styles as needed.
   - Modify `assets/js/script.js` for any interactive functionality.

4. **Add Images**
   Place any images you want to use in the `images` directory.

5. **Fonts**
   Add any custom fonts to the `assets/fonts` directory to ensure typography matches the design.
